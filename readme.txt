# Luxu’s – Development Preview

This repository hosts the development preview of the Luxu’s project.  
The current build is published via GitHub Pages for the purpose of **internal testing**,  
UI verification, and gameplay behavior checks.

Please note:

- The codebase is **under active development** and does not represent the final product.
- Features, logic, and assets may change without notice.
- External code reviews, forks, or reuse of the source are **not within the scope** of this preview.
- Issues and pull requests are currently **not being accepted**, as this repository is used solely for internal iteration.

If you are viewing this repository, please treat it as a **temporary staging environment**  
rather than a public open-source project.

Thank you for your understanding.
