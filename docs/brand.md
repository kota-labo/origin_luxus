# Luxu's ブランド定義

> このドキュメントは Luxu's の価値観・設計思想・ブランドメッセージを定義する。
> UI実装・コピーライティング・機能優先度の判断軸として参照すること。

---

## ブランドの本質

Luxu's は、派手な演出やカジノ的な刺激ではなく、
**「静かに深い分析を楽しむ」** という、ポーカーの本質に寄り添った知的な体験を提供するブランド。

---

## 3つの価値の柱

### 1. Minimal（ミニマル）— 無駄を削ぎ落とした集中体験

- 余白を活かした白×黒のUI
- 必要な情報だけが静かに表示される
- アニメーションは最小限
- どのゲーム（NLH / 2-7 / Badugi）でも統一された操作感

→ プレイヤーは "ゲームそのもの" に集中できる。

**ミニマルは「簡素」ではなく、"本質だけを残す" という贅沢さを表現している。**

---

### 2. Intellectual（知的）— 深い理解と成長を促す設計

Luxu's の最大の価値は、プレイヤーの思考を深め、上達を可視化すること。

将来的に提供する分析機能（ロードマップ）:
- ハンド履歴
- ドロー傾向の分析
- Badugi枚数の推移
- 2-7 のハンド強さの分布
- CPUレベル別の勝率
- プレイスタイル診断（タイト/ルース/ドロー傾向など）
- Mix Game 総合評価

これらは単なる数字ではなく、**"自分のポーカーが見える" という知的快感** を生む。

Luxu's がデザインするサイクル:
```
プレイする → 振り返る → 気づく → 上達する
```

---

### 3. Luxury（高級感）— 静かで洗練された体験

Luxu's の高級感は、派手な金色やカジノ演出ではなく、**"静かで洗練された美しさ"** に宿る。

- 白×黒のコントラスト
- 滑らかなカードアニメーション
- 余白のあるタイポグラフィ
- 統一されたUIコンポーネント
- 美しい統計グラフ（線・棒・円のミニマルデザイン）

**高級感とは「派手さ」ではなく、"丁寧に作られたものだけが持つ静かな存在感" のこと。**

---

## ブランドの存在意義（Why Luxu's）

| 価値 | 内容 |
|------|------|
| **学びの場** | 2-7 / Badugi のようなニッチゲームを学べる場所はほぼない。Luxu's がその唯一の場を提供する。 |
| **成長の可視化** | ただ遊ぶだけでなく「自分のプレイがどう変わったか」が見える。継続率を高める設計。 |
| **集中できる環境** | 広告なし、派手な演出なし。カードと自分の思考だけがある。大人のためのポーカーアプリ。 |
| **Mix Game の入り口** | Mix Game は複雑だが、UI・分析・統計を通して「理解しながら楽しめる」体験を提供する。 |

---

## ブランドメッセージ（コピー候補）

### 1行メッセージ

- 静かに、深く、ポーカーを楽しむ。
- 思考が磨かれるポーカー体験。
- 余白が生む、集中のポーカー。
- 知性を満たす、ミニマルなポーカー。

### ブランドステートメント

> Luxu's は、ミニマルで洗練されたデザインの中に、
> ポーカーの深い思考と成長を宿すブランドです。
> 派手な演出ではなく、静かな分析と美しい統計を通して、
> あなたのポーカーをより知的に、より豊かにします。

---

## 設計上の判断基準

この定義は、機能追加・UI変更・コピーライティングを判断する際の軸として使う。

| 判断軸 | OK | NG |
|--------|-----|-----|
| アニメーション | 必要最小限・滑らか | 過剰・ゲーミング的 |
| 情報表示 | 必要な情報だけ静かに | 情報過多・常時表示 |
| デザイン | 余白・コントラスト・統一感 | 派手な色使い・過多な装飾 |
| 機能追加 | 分析・学習・思考深化 | 課金誘導・派手演出・ランキング競争 |
| コピー | 知的・落ち着き・本質 | カジノ的・煽り・興奮訴求 |
